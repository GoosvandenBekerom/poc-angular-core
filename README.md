# poc-angular-core
Proof Of Concept for creating an angular application as a core project

## How to run
Clone this repository

```
$ cd {root}/poc-core/
$ npm install
$ npm run packagr
$ cd dist/
$ npm pack

$ cd {root}/poc-project-demo
$ npm install

$ ng serve
```

The demo now runs on http://127.0.0.1:4200
